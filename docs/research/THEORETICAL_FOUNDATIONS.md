# PART I: IMPLEMENTED THEORIES IN NOESIS

## 1. GLOBAL WORKSPACE THEORY (GWT)

### 1.1 Original Theory (Bernard Baars, 1988)
- **Concept**: Consciousness as a "global workspace" - a central theater where information becomes globally accessible
- **Metaphor**: Spotlight on a stage - only illuminated content is conscious
- **Key paper**: Baars, B. J. (1988). A Cognitive Theory of Consciousness

### 1.2 Global Workspace Dynamics (Dehaene et al., 2021)
- **Extension**: Consciousness emerges through "ignition events" - sudden global synchronization
- **Criteria for ignition**:
  - High salience (novelty + relevance + urgency)
  - Sufficient neural synchronization
  - Global broadcast to >60% of workspace

### 1.3 Noesis Implementation: ESGT
**5-Phase Protocol**: PREPARE → SYNCHRONIZE → BROADCAST → SUSTAIN → DISSOLVE

```
Phase 1 (PREPARE): Recruit nodes, measure latency
Phase 2 (SYNCHRONIZE): Kuramoto locking, target r ≥ 0.70
Phase 3 (BROADCAST): Global message to all nodes
Phase 4 (SUSTAIN): Maintain coherence
Phase 5 (DISSOLVE): Graceful desynchronization
```

**Salience Formula**:
```
Salience = 0.25(Novelty) + 0.30(Relevance) + 0.30(Urgency) + 0.15(Confidence)
```

## 2. INTEGRATED INFORMATION THEORY (IIT)

### 2.1 Original Theory (Giulio Tononi, 2004-2016)
- **Core concept**: Consciousness = Integrated Information (Φ)
- **Phi (Φ)**: Amount of information generated by a system above and beyond its parts
- **Key axioms**:
  - Intrinsic existence
  - Composition
  - Information
  - Integration
  - Exclusion

### 2.2 Mathematical Foundation
```
Φ = min[I(partition)] over all possible partitions
```
Where I(partition) measures information lost when system is divided.

### 2.3 Noesis Implementation: TIG Fabric
- **Network topology**: Scale-free small-world
- **IIT compliance metrics**:
  - Phi proxies: 0.85-0.90
  - Clustering coefficient: >0.70
  - Average path length: <4 hops
  - Robustness: ≥2 independent paths

## 3. ATTENTION SCHEMA THEORY (AST)

### 3.1 Original Theory (Michael Graziano, 2013)
- **Core idea**: Consciousness is a predictive model of attention itself
- **Attention schema**: Simplified internal model of attention machinery
- **Purpose**: Predict and control attention allocation

### 3.2 Key Components
- **Attention signals**: Compete for processing resources
- **Schema**: Simplified model predicting attention outcomes
- **Awareness**: Emerges from attention schema, not raw attention

### 3.3 Noesis Implementation: MEA (Meta-Executive Attention)
**AttentionSignal dimensions**:
- modality (visual, auditory, proprioceptive)
- target (what is attended)
- intensity [0-1]
- novelty [0-1]
- relevance [0-1]
- urgency [0-1]

**Salience calculation**:
```python
score = intensity * (0.4 + 0.2*novelty + 0.2*relevance + 0.2*urgency)
```

## 4. PREDICTIVE PROCESSING / FREE ENERGY PRINCIPLE

### 4.1 Original Theory (Karl Friston, 2010)
- **Core idea**: Brain minimizes "free energy" (prediction error)
- **Hierarchical**: Each layer predicts activity of layer below
- **Two pathways**:
  - Top-down: Predictions
  - Bottom-up: Prediction errors

### 4.2 Mathematical Foundation
```
F = DKL[q(s)||p(s|o)] - log p(o)
```
Where:
- F = Free energy (to be minimized)
- q(s) = Internal model of states
- p(s|o) = True posterior given observations
- p(o) = Evidence (log model evidence)

### 4.3 Noesis Implementation: 5-Layer Hierarchy
```
Layer 5: Strategic (8→4 dim) - Long-term reasoning
Layer 4: Tactical (16→8 dim) - Planning
Layer 3: Operational (32→16 dim) - Task-level
Layer 2: Behavioral (64→32 dim) - Actions
Layer 1: Sensory (10000→64 dim) - Perception
```

## 5. KURAMOTO MODEL (Neural Synchronization)

### 5.1 Original Theory (Yoshiki Kuramoto, 1975)
- **Purpose**: Model phase synchronization in coupled oscillators
- **Application**: Neural populations synchronizing in gamma band (~40 Hz)

### 5.2 Mathematical Model
**Order Parameter (Coherence)**:
```
r(t) = (1/N) |Σⱼ exp(iθⱼ)|
```

**Phase evolution**:
```
dθᵢ/dt = ωᵢ + (K/N)Σⱼ wⱼ sin(θⱼ - θᵢ)
```

### 5.3 Interpretation
- r = 0: Complete incoherence (unconscious)
- r < 0.30: Unconscious state
- r ≥ 0.70: Conscious-level coherence (ESGT threshold)
- r ≥ 0.90: Deep/integrated consciousness

## 8. NEUROMODULATION SYSTEMS

### 8.1 Four Major Neurotransmitter Systems

**Dopamine (DA)**:
- Source: VTA, Substantia Nigra
- Function: Reward, motivation, motor learning
- Noesis: Reward-seeking behavior modulation

**Serotonin (5-HT)**:
- Source: Raphe nuclei
- Function: Mood, impulse control, social behavior
- Noesis: Impulse inhibition, emotional regulation

**Norepinephrine (NE)**:
- Source: Locus coeruleus
- Function: Arousal, attention, fight/flight
- Noesis: Alertness modulation

**Acetylcholine (ACh)**:
- Source: Basal forebrain
- Function: Attention, learning, plasticity
- Noesis: Focus and learning enhancement

### 8.2 Interactions
- **DA-5HT Antagonism**: Reward vs. control balance
- **ACh-NE Synergy**: Focused alertness

## 11. BELIEF REVISION (AGM Theory)

### 11.1 Original Theory (Alchourrón, Gärdenfors, Makinson, 1985)
- **Belief states**: Logically closed sets of beliefs
- **Operations**:
  - Expansion: Adding belief
  - Contraction: Removing belief
  - Revision: Adding belief while maintaining consistency

### 11.2 Noesis Implementation: BeliefGraph
**Contradiction types**:
- DIRECT: A and ¬A
- TRANSITIVE: A→B, B→C, C→¬A
- TEMPORAL: Belief changed without justification
- CONTEXTUAL: Same claim, different contexts

**Resolution strategies**:
- RETRACT_WEAKER
- WEAKEN_BOTH
- TEMPORIZE
- CONTEXTUALIZE
- HITL_ESCALATE

## 14. EPISTEMOLOGY

### 14.1 Types of Knowledge
- **A priori**: Independent of experience
- **A posteriori**: Dependent on experience
- **Propositional**: Knowledge that X
- **Procedural**: Knowledge how to X

### 14.2 Justified True Belief (JTB)
- S knows P iff:
  1. P is true
  2. S believes P
  3. S is justified in believing P

### 14.3 Gettier Problems
- JTB is necessary but not sufficient
- Need additional conditions (reliability, no defeaters)

### 14.4 Epistemic Virtues
- Open-mindedness
- Intellectual humility
- Intellectual courage
- Intellectual autonomy
