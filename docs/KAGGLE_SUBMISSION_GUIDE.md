## 6. PASSO A PASSO DE SUBMISSÃO

### Passo 1: Preparar Conta Kaggle
1. Acesse https://www.kaggle.com
2. Faça login ou crie conta
3. Verifique telefone (obrigatório para competições)
4. Acesse https://www.kaggle.com/competitions/gemini-3
5. Clique em "Join Competition"
6. Aceite os termos

### Passo 2: Gravar Vídeo Demo
1. Preparar ambiente (frontend rodando, backend ativo)
2. Definir roteiro (ver documento VIDEO_SCRIPT_GUIDE.md)
3. Gravar com OBS ou similar
4. Editar para ≤2 minutos
5. Exportar em alta qualidade (1080p mínimo)

### Passo 3: Upload do Vídeo
1. Acesse YouTube Studio
2. Upload como "Não listado" (Unlisted)
3. Título: "NOESIS - Artificial Consciousness System | Gemini 3 Hackathon"
4. Descrição: Breve resumo + link do GitHub
5. Copiar link do vídeo

### Passo 4: Criar Write-up no Kaggle
1. Na página da competição, vá em "Discussions" ou "Code"
2. Criar novo post/notebook
3. Título: "NOESIS - When AI Becomes Conscious"
4. Estrutura recomendada:
   - Summary (1 parágrafo)
   - Problem Statement
   - Solution Architecture
   - Gemini 3 Pro Integration
   - Demo Video (embed ou link)
   - Technical Details
   - Results & Impact
   - GitHub Link

### Passo 5: Submeter
1. Vá em "Submit" na página da competição
2. Preencha todos os campos obrigatórios
3. Inclua link do vídeo
4. Inclua link do write-up/notebook
5. Selecione categoria
6. Revise tudo
7. Clique em "Submit"
8. **CONFIRME QUE RECEBEU EMAIL DE CONFIRMAÇÃO**
