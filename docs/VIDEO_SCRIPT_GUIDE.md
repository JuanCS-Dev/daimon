## FLUXO DE GRAVAÇÃO RECOMENDADO

### Preparação:
1. Frontend rodando (`npm run dev`)
2. Backend ativo (`./noesis wakeup`)
3. Memória limpa ou com contexto apropriado
4. Tela em 1080p ou superior
5. OBS ou gravador configurado

### Durante gravação:
1. Gravar cada seção separadamente (facilita edição)
2. Fazer 2-3 takes de cada
3. Pausar entre seções
4. Verificar áudio após cada take

### Pós-produção:
1. Selecionar melhores takes
2. Cortar silêncios e erros
3. Adicionar transições suaves
4. Incluir textos/overlays
5. Ajustar áudio
6. Exportar em alta qualidade
