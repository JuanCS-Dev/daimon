## How to Become a Contributor

We welcome contributions from anyone interested in artificial consciousness! See our [Contributing Guide](CONTRIBUTING.md) for details.

### Ways to Contribute

1. **Code**: Fix bugs, add features, improve performance
2. **Documentation**: Improve docs, add examples, fix typos
3. **Testing**: Add tests, report bugs, verify fixes
4. **Research**: Propose consciousness theories, analyze architectures
5. **Community**: Answer questions, review PRs, help others

### Recognition

Contributors are recognized in:
- This file (CONTRIBUTORS.md)
- Release notes
- Project documentation
